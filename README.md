# Valentine's Next

A Next.js application built for a Valentine's Day proposal.

## Features

- **Interactive Proposal**: A playful "Will you be my Valentine?" page with dynamic "Yes" and "No" buttons. The "No" button runs away/shrinks, while the "Yes" button grows!
- **Mobile Responsive**: Fully optimized for mobile devices.
- **Yay Celebration Page**:
  - Custom Togepi GIF.
  - Interactive Envelope component to read the letter.
  
  
## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the application**:
   Visit [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework**: Next.js
- **Styling**: Global CSS & CSS Modules
- **Language**: JavaScript / React

## Project Structure

- `app/page.js`: Main proposal page.
- `app/yay/page.js`: Acceptance/Success page.
- `components/Envelope.js`: Interactive letter component.
- `app/globals.css`: Global styles and mobile optimizations.
